Best practices for starting a Pelican site

Published: Sun 03 October 2010
Updated: Mon 04 October 2010
By Alexis Metaireau Conan Doyle

In yeah.

tags: thats awesome

  1. Create a new git repository at Github: https://github.com/new

    • Add Python to .gitignore

  2. Clone the newly created repository:

    $ git clone <repo_https_clone_url> <site_root>

  3. Create virtualenv (command only works if you have virtualenvwrapper):

    $ mkvirtualenv <venv_name>

  4. Install pelican (if you're not confident with reStructuredText yet, then install Markdown as well):

    $ pip install pelican markdown

  5. Navigate to <site_root> and execute the pelican-quickstart script:

    $ cd <site_root>
$ pelican-quickstart
